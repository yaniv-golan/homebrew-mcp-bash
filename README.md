# homebrew-mcp-bash

Homebrew tap for [mcp-bash](https://github.com/yaniv-golan/mcp-bash-framework) — MCP server framework for Bash.

## Installation

```bash
brew install yaniv-golan/mcp-bash/mcp-bash
```

Or tap first, then install:

```bash
brew tap yaniv-golan/mcp-bash
brew install mcp-bash
```

## Updating

```bash
brew upgrade mcp-bash
```

## About

mcp-bash lets you build [Model Context Protocol](https://modelcontextprotocol.io) servers using plain Bash scripts.
See the [main repository](https://github.com/yaniv-golan/mcp-bash-framework) for full documentation.
